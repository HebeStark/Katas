# Descripción

Es la base de un juego de memoria para encontrar pares de cartas iguales.

En esta primera parte se implementa únicamente el -Tablero del juego

El tablero muestra las cartas boca abajo y un marcador de puntos inicializado en 0.

## Tecnologías utilizadas

-HTML5
-CSS3
-CSS Grid

## Estructura del tablero

El tablero está compuesto por:

-4 columnas
-4 filas
-Total 16 cartas

Las cartas se muestran boca abajo utilizando una imagen de reverso.

## Funcionalidades actuales

-Visualización del tablero de juego
-Distribución de cartas en una cuadrícula 4x4
-Cartas representadas boca abajo
-Marcador de puntos inicial (0 puntos)
