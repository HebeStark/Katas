# Juego de Memoria

Pequeño juego de memoria desarrollado con HTML, CSS y JavaScript.
El objetivo es encontrar todas las parejas de cartas en el menor tiempo posible.

## Cómo jugar

1-Haz clic en una carta para descubrir su imagen.
2-Haz clic en una segunda carta para intentar encontrar su pareja.
3-Si las cartas coinciden:
Se sumará 1 punto.
Las cartas quedarán bloqueadas en el tablero.
4-Si no coinciden:
Las cartas volverán a su reverso después de un segundo.
El juego termina cuando encuentras los 8 pares de cartas.

## Funcionalidades

El juego incluye:
-Tablero de memoria 4x4
-Cartas mezcladas aleatoriamente
-Contador de selecciones
-Contador de tiempo en segundos
-Detección automática de victoria
-Bloqueo de cartas correctas
-Botón para reiniciar la partida

## Tecnologías utilizadas

HTML5 → estructura de la página
CSS3 → diseño y estilos del tablero
JavaScript (Vanilla) → lógica del juego y manipulación del DOM

## Cómo ejecutar el proyecto

Abrir el archivo index.html en el navegador.
No requiere instalación ni dependencias adicionales.
